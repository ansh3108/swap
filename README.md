# Smart Swap

---

### Durability Protection

Never accidentally destroy a high level Mending tool again. It constantly monitors your item's durability and If the tool drops to ** 2 durability**, it automatically swaps it for a fresh tool from your inventory before it shatters.

### Intelligent Fuzzy Matching
You inventory doesn't have to be perfectly organized!
`ItemTags` understands item categories:

* **Tools:** A broken Diamond Pickaxe will automatically fall back to an Iron Pickaxe if that is what you have available.
  <img width="800" height="342" alt="pickaxe" src="https://github.com/user-attachments/assets/ed484650-587c-4083-aa4c-991d9a4beeaa" />

* **Blocks:** If you exhaust your stack of spruce planks, It will swap to Oak Planks.
  
<img width="800" height="342" alt="planks" src="https://github.com/user-attachments/assets/f85c4d0e-2a8c-45cb-8bb9-8f769e2edb81" />


### Kill Switch
Press `V` (rebindable) to toggle swap logic on or off.
